# Яндекс практикум веб-разработка
## Проектная работа 2 "Научиться учиться"

Одностраничный сайт для закрепления теории, полученной в тренажере.

Использованы технологии:
1. HTML
2. CSS
3. CSS анимация
4. Flexbox
5. Nested БЭМ
